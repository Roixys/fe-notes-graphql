# Notes App Frontend - Ahmadhi

This is the frontend for the Notes App, built using Next.js with TypeScript, Apollo Client, Chakra UI, and GraphQL. The app allows users to perform CRUD operations on notes.

## Technologies Used

- [Next.js](https://nextjs.org/) - A React framework for server-rendered applications.
- [TypeScript](https://www.typescriptlang.org/) - A typed superset of JavaScript.
- [Apollo Client](https://www.apollographql.com/docs/react/) - A GraphQL client for React.
- [Chakra UI](https://chakra-ui.com/) - A simple, modular, and accessible component library for React.
- [GraphQL](https://graphql.org/) - A query language for your API.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your system.
- [Node.js](https://nodejs.org/) for development purposes.

### Development

1. **Clone the Repository**

```bash
git clone git@github.com:Roixys/fe-notes-graphql.git
cd fe-notes-graphql
```

2. **Install Dependencies**

```bash
bun install
```

3. **Run development server**

```shell
bun dev
```

The server will run on port 3000.
